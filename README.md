# Angular Admin Demo

This is a simple demo project.

## Dependencies

Running the demo locally requires Node and the Angular command-line interface (CLI). 

1. Download and install [Node](https://nodejs.org/en/download/) for your operating system
2. In the command line, run `npm install -g @angular/cli` to install the Angular CLI

## Installing and running locally

Run these commands from a terminal:

```bash
git clone git@github.com:bitovi/angular-admin-demo.git
cd angular-admin-demo
npm install
ng serve
```

Open a browser window and navigate to `http://localhost:4200/` to see the application.
